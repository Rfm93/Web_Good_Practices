
Code example to demonstrate the 3 ways to add JavaScript to HTML and demonstrate good practices in web development, separating structure, style and behavior.

The 3 ways to add CSS or JS to HTML: inline, internal and external.

The first two present readability and code organization problems, affecting scalability.

In addition, parallel loading of files speeds up page loading and the CSS and JS are stored in the browser cache, avoiding having to download the files again.
